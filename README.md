N-Stock - Sistema de Gestión de Inventario y Ventas

Descripción del Proyecto N-STOCK es un sistema web cloud de gestión de inventario y punto de venta, orientado a Pymes del sector retail. Permite el control centralizado de stock, ventas y reposición operando bajo un modelo de múltiples sucursales con control de acceso por roles (Administrador y Cajero).

Tecnologías Utilizadas

Backend: Java 21, Spring Boot, JPA/Hibernate.
Base de Datos: MySQL (Alojada en la nube mediante Aiven).
Frontend: HTML5, CSS3, JavaScript.
Infraestructura / DevOps: Despliegue continuo en la nube a través de Render.
